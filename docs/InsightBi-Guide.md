## 3. 홈 (Smart Home)

**경로:** `/`

### 주요 기능

#### AI 검색
- 자연어로 데이터 질문 입력
- 자동으로 SQL 생성 → 데이터 조회 → 차트/테이블 결과 표시
- SQL 보기 토글, 👍/👎 피드백 제출 가능

#### 인사이트 브리핑
- NPL·VaR·LCR 핵심 지표 자동 요약
- 현재 날짜 기준 실시간 표시

#### 빠른 시작
| 버튼 | 이동 |
|------|------|
| 새 질문 | `/questions/pick` |
| 새 대시보드 | `/builder` |
| 데이터 탐색 | `/browse` |
| 컬렉션 | `/collections` |

#### 즐겨찾기 (Pinned)
- 항목 카드에서 ★ 클릭 → 홈 즐겨찾기 등록
- 홈에 그리드로 표시, 재클릭으로 해제

#### 최근 방문
- 페이지 이동 시 자동 기록 (최대 8개)
- 방문 시간 표시 (몇 분 전, 몇 시간 전 등)

## 4. 리스크 관리

### 4-1. 신용리스크 `/credit-risk`

|----|------|------|

## 5. 분석 메뉴

### 5-1. 질문 (NotebookEditor) `/questions/new`

SQL을 모르는 사용자를 위한 단계별 질문 생성기.

```
[Step 1] 데이터셋 선택 → [Step 2] 필터 설정 → [Step 3] 차트 타입 선택 → 실행 → 저장
```

| 단계 | 설명 |
|------|------|
| 데이터셋 선택 | 15개+ 데이터셋 카테고리별 목록에서 선택 |
| 필터 설정 | 컬럼 + 연산자(=, ≥, ≤, 포함) + 값 복수 조합 |
| 차트 타입 | 라인/면적/막대/파이 등 |
| 저장 | 질문 저장 → 컬렉션 추가 옵션 |

---

### 5-2. 대시보드 목록 `/dashboards`

저장된 커스텀 대시보드 갤러리 페이지.

| 기능 | 설명 |
|------|------|
| 열기 | `/builder?load=이름` 으로 워크스페이스에서 바로 로드 |
| 홈 설정 | 해당 대시보드를 홈 화면으로 지정 |
| 삭제 | 대시보드 영구 삭제 |
| 새로고침 | 서버/localStorage 최신 목록 반영 |

---

### 5-3. 워크스페이스 `/builder`

드래그앤드롭 기반 커스텀 대시보드 빌더.

#### 편집 모드
1. 좌측 데이터 카탈로그에서 데이터셋 `+` 클릭 → 캔버스에 위젯 추가
2. 위젯 드래그로 위치 이동
3. ⚙ 클릭 → 차트 타입 / 축 설정 / 임계치 설정 (MappingPanel)
4. 크기 조절: 우측 하단 모서리 드래그 (1~3칸)
5. 저장 → 대시보드 라이브러리에 추가 + 컬렉션 연동

#### 뷰어 모드 (기본)
- 저장된 위젯 읽기 전용 표시
- **자동 새로고침:** 1분 / 5분 / 10분 / 30분 선택 가능
- **미리보기** 버튼으로 저장 결과 확인

#### 전역 필터 (FilterBar)
- 날짜 범위 / 부서 선택 → 전체 위젯에 동시 적용

#### 역할별 권한
| 역할 | 편집 | 저장 | 초기화 |
|------|:----:|:----:|:------:|
| Admin | ✅ | ✅ | ✅ |
| Editor | ✅ | ✅ | ❌ |
| Viewer | ❌ | ❌ | ❌ |

## 6. 탐색 (Browse)

**경로:** `/browse` → 데이터셋 목록
**경로:** `/browse/[datasetId]` → 데이터셋 상세

### 데이터셋 목록 페이지
- 신용·시장·유동성·NCR 카테고리별 카드 표시
- 데이터셋 클릭 → 상세 페이지 이동

### 데이터셋 상세 페이지

| 섹션 | 내용 |
|------|------|
| **통계 카드** | 총 행수 / 컬럼 수 / 측정값 수 / 날짜 컬럼 수 |
| **필터 패널** | 컬럼 + 연산자 + 값 → 실시간 데이터 필터링 |
| **차트 미리보기** | 막대/선/면적/파이 타입 전환 가능 |
| **컬럼 통계** | 측정값 컬럼별 min / avg / max |
| **스키마** | 컬럼명·타입·역할·단위 목록 |
| **데이터 미리보기** | 최대 20행 테이블 (컬럼 클릭으로 정렬) |

**액션 버튼:**
- `질문 만들기` → `/questions/new`
- `대시보드에 추가` → `/builder`

|------|------|

### 8-2. NoCode 비주얼 쿼리 빌더 `/questions/nocode`

쿼리를 모르는 사용자도 클릭만으로 데이터 분석이 가능한 Metabase 스타일 빌더.

#### 구성 섹션

**① 데이터 섹션** (파란 영역)
```
[ ⠿ NPL 추이 (신용리스크) ▼ ]                    [ ▶ ]
[ ⊙ 원시 데이터 ]  [ ⠿ 요약 ]
```
- 테이블 칩 클릭 → 데이터셋 변경 피커 펼침
- `▶` 버튼 또는 하단 시각화 버튼으로 실행
- **원시 데이터:** 필터만 적용한 원본 행 표시
- **요약:** 집계 함수 + 그룹화 적용

**② 필터 섹션** (회색 영역)
```
[ 컬럼 선택 ]  [ = / ≥ / ≤ / 포함 ]  [ 값 입력 ]  [X]
[ + 답변을 좁히기 위해 필터 더하기 ]
```
- 필터 복수 추가 가능
- X 버튼으로 개별 제거

**③ 요약 섹션** (초록 영역, 요약 모드일 때만 표시)
```
[ COUNT / SUM / AVG / MIN / MAX ]  [ 메트릭 컬럼 ]  (으)로  [ 그룹화 열 선택 ]
```
| 함수 | 설명 |
|------|------|
| COUNT | 행 개수 |
| SUM | 합계 |
| AVG | 평균 |
| MIN | 최솟값 |
| MAX | 최댓값 |

**④ 정렬 / 행수 제한**
- 정렬: 컬럼 선택 + 오름차순/내림차순
- 행수 제한: 최대 N행까지만 조회

**⑤ 시각화 버튼**
```
[              시각화              ]
```

**⑥ 결과 영역**
- 차트 타입 전환: 막대 / 선 / 파이 / 표
- `저장` 버튼 → 컬렉션 추가 연동

## 9. 우측 액션 버튼

| 버튼 | 기능 |
|------|------|
| 🤖 **AI 분석** | 우측 패널 슬라이드 열기 — 자연어 질문, 대화 이력 유지, SQL 확인, 피드백 |
| 🛡 **관리자** | AI 학습 데이터 관리 (로그인 필요) |
| 🔔 **알림** | 미읽음 리스크 알림 뱃지 표시 |
| ☀/🌙 **테마** | 라이트 / 다크 모드 전환 |
| 👤 **역할** | Admin / Editor / Viewer 전환 |

### AI 패널 상세
- 자연어 입력 → SQL 자동 생성 → 결과 차트
- 대화 이력 유지 (세션 내)
- 👍/👎 피드백 → 관리자 검토 후 AI 재학습 반영
- Smart Narrative: AI가 결과를 자동으로 설명
